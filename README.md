# ESpeech-Gradio

Локальный интерфейс на базе **Gradio** для высококачественного синтеза русской речи с использованием модели **F5-TTS**, обученной на масштабном корпусе **ESpeech**.

## О проекте ESpeech

ESpeech — это новый крупномасштабный корпус русской речи, включающий более **4700 часов** аудио. Корпус собран из различных доменов, таких как подкасты и образовательные вебинары, что обеспечивает богатое разнообразие стилей речи.

### Ключевые особенности корпуса:
- **Разнообразие**: Данные из "дикой природы" (in-the-wild), включая спонтанную речь в подкастах и структурированные монологи в вебинарах.
- **Качество**: Многоэтапный конвейер обработки данных, включающий разделение источников, диаризацию спикеров, сегментацию VAD и строгую фильтрацию на основе композитной оценки MOS (Mean Opinion Score).
- **Сингл-спикер датасеты**: В рамках проекта выпущено четыре высококачественных датасета отдельных дикторов: `igm`, `buldjat`, `upvote` и `tuchniyzhab`.

## Возможности приложения

- **Zero-shot TTS**: Синтез речи на основе короткого референсного аудио (до 12 секунд). Модель способна имитировать голос, темп и эмоции диктора.
- **Интеллектуальные ударения (RUAccent)**:
  - **Auto**: Автоматическая расстановка ударений.
  - **Manual**: Использование только ваших меток ударения (символ `+` перед гласной, например: `прив+ет`).
  - **Hybrid**: Сочетание автоматики и ручных правок.
- **Гибкие настройки**:
  - Управление количеством шагов диффузии (NFE).
  - Регулировка скорости речи.
  - Настройка плавности переходов (Cross-Fade).
  - Удаление тишины из итогового аудио.
- **Пресеты**: Возможность быстрого выбора заранее подготовленных голосов.

## Установка

### Требования
- Python 3.11 или выше.
- [uv](https://github.com/astral-sh/uv) (рекомендуемый менеджер пакетов).

### Инструкция

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd espeech
   ```

2. Создайте виртуальное окружение и установите зависимости:
   ```bash
   uv sync
   ```

## Запуск

Для запуска веб-интерфейса используйте команду:

```bash
uv run espeech
```

После запуска интерфейс будет доступен по адресу: `http://127.0.0.1:7860`.

> [!NOTE]
> Первый запуск может занять некоторое время, так как модель (`ESpeech-TTS-1_RL-V2`), вокодер и словари для ударений будут загружены автоматически из Hugging Face.

## Использование

1. **Загрузите референс**: Перетащите аудиофайл с чистым голосом (без шума и музыки) и введите его транскрипцию.
2. **Введите текст**: Напишите текст, который нужно озвучить.
3. **Подготовьте текст**: Нажмите кнопку "Preview text", чтобы увидеть, как расставлены ударения.
4. **Синтезируйте**: Нажмите "Generate speech" для получения результата.
